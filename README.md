# Ecofood
O EcoFood é uma aplicação web desenvolvida com o objetivo de reduzir o desperdício de alimentos, ajudando usuários a gerenciar seus mantimentos e tomar decisões mais sustentáveis.

Objetivo de Desenvolvimento Sustentável (ODS)
Este projeto está alinhado com o objetivo da ONU:
ODS 12 – Consumo e Produção Responsáveis

Problema
Muitas pessoas desperdiçam alimentos por falta de controle sobre:
itens disponíveis em casa
datas de validade
planejamento de consumo
Gerandoi
prejuízo financeiro
aumento de resíduos
impacto ambiental

Solução
UM sistema que permite:
Cadastro de alimentos
Controle de validade
Alertas de vencimento
Sugestões de receitas com base nos itens disponíveis
Relatórios de desperdícios
Objetivo: ajudar o usuário a consumir de forma mais consciente e sustentável.

Tecnologias (planejadas)
As seguintes tecnologias serão utilizadas no desenvolvimento:
Frontend: HTML, CSS e JavaScript
Backend: Node.js
Banco de Dados: MongoDB
Controle de versão: Git e GitHub
